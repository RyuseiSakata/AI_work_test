# Antigravity WordPress自動投稿システム 🚀

このシステムは、外部のAI API（Make/OpenAIなど）を使わず、**Antigravity自身がAIとなり、記事作成・画像生成・WordPressへの入稿を全自動で行う**ためのツール群です。

## 📁 ファイル構成

*   `wp_auto_poster.py`: Antigravityが呼び出して、WordPressのAPIに記事と画像をアップロードするためのPythonプログラム。
*   `.env.example`: パスワード設定ファイルのひな形。

---

## ⚙️ 初回セットアップ手順

パスワードなどを安全に管理するため、設定ファイルを作成します。

1. このフォルダの中にある `.env.example` というファイルをコピーし、名前を **`.env`**（ドットエンブ）に変更してください。
2. 作成した `.env` ファイルをテキストエディタで開き、以下の情報をあなたの環境に合わせて書き換えて保存します。

```text
WP_URL=https://your-domain.com
WP_USER=admin
WP_APP_PASS=xxxx xxxx xxxx xxxx xxxx xxxx
```

※ `WP_APP_PASS` はWordPress管理画面の「ユーザー」＞「プロフィール」の一番下から発行できるアプリケーションパスワードです。

---

## 🧑‍💻 使い方（Antigravityへの依頼方法）

設定が終わったら、以降の操作は**あなたからAntigravity（私）にチャットでお願いするだけ**です！

### 依頼用テンプレート
以下の文章をコピーして、イベント内容を埋めて私（Antigravity）にチャットで送信してください。

> **以下のイベント情報をもとに、WordPressに記事を下書き投稿して！**
> 
> ・イベント名：春のモデルハウス見学会
> ・開催日時：4月XX日(土)〜XX日(日)
> ・場所：〇〇県〇〇市
> ・ターゲット：30代の子育て世帯
> ・アピールポイント：広い土間収納、家事動線、無垢材
> ・画像のヒント：ナチュラルテイストで明るいリビング

これを送っていただければ、私が内部的に以下の処理をすべて代行します。
1. ブログ用のHTML文章の生成
2. ターゲットとヒントに合わせたアイキャッチ画像の生成
3. `wp_auto_poster.py` を使ってWordPressへ下書き投稿

## 🛠️ トラブルシューティング
*   **「エラー: .envファイルに...」と出る場合:** `.env` ファイルが正しく作成・保存されているか確認してください。
*   **「ステータスコード: 401」と出る場合:** ユーザー名やアプリケーションパスワードが間違っていないか確認してください。
*   **「ステータスコード: 403」と出る場合:** セキュリティプラグイン（SiteGuardなど）で REST API が制限されていないか確認してください。
