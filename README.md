# Antigravity WordPress自動投稿システム 🚀

このシステムは、外部のAI API（Make/OpenAIなど）を使わず、**Antigravity自身がAIとなり、記事作成・画像生成・WordPressへの入稿を全自動で行う**ためのツール群です。

## 📁 ファイル構成

*   `wp_auto_poster.py`: Antigravityが呼び出して、WordPressのAPIに記事と画像をアップロードするためのPythonプログラム。
*   `.env.example`: パスワード設定ファイルのひな形。

---

## ⚙️ 初回セットアップ手順

パスワードなどを安全に管理するため、設定ファイルを作成します。

1. このフォルダの中にある `.env.example` というファイルをコピーし、名前を **`.env`**（ドットエンブ）に変更してください。
2. 作成した `.env` ファイルをテキストエディタで開き、以下の情報をあなたの環境に合わせて書き換えて保存します。

```text
WP_URL=https://your-domain.com
WP_USER=admin
WP_APP_PASS=xxxx xxxx xxxx xxxx xxxx xxxx
```

※ `WP_APP_PASS` はWordPress管理画面の「ユーザー」＞「プロフィール」の一番下から発行できるアプリケーションパスワードです。

---

## 🧑‍💻 使い方（Antigravityへの依頼方法）

API (`wp_auto_poster.py`) を使用した自動投稿と、**AIエージェントモード（ブラウザ自動操作）**を用いた投稿の２つの方法があります。

---

### 方法1：AIエージェントモード（ブラウザ自動操作）を使った投稿 🌟おすすめ
WordPressのセキュリティ制限（REST APIの制限など）を回避して、人間と同じようにブラウザ画面を自動操作して確実に入稿できる方法です。

**【依頼用テンプレート】**
以下の文章をコピーし、内容を埋めてAntigravityに送信してください。

> **以下の情報をもとに、ブログ用の文章と画像を生成し、AIエージェントモード（ブラウザ操作）を使ってWordPressに下書き保存してください。**
> 
> [ログイン情報]
> ・WordPressログインURL：https://your-domain.com/wp-admin
> ・ユーザー名：[あなたのユーザー名]
> ・パスワード：[あなたのログインパスワード]
> 
> [記事情報]
> ・イベント名：春のモデルハウス見学会
> ・開催日時：4月XX日(土)〜XX日(日)
> ・場所：〇〇県〇〇市
> ・ターゲット：30代の子育て世帯
> ・アピールポイント：広い土間収納、家事動線、無垢材
> ・画像のヒント：ナチュラルテイストで明るいリビング

---

### 方法2：API (`wp_auto_poster.py`) を使った投稿
事前の `.env` 設定を使ってバックグラウンドで投稿を送信します。（※セキュリティプラグイン等ではじかれる場合があります）

**【依頼用テンプレート】**
> **以下のイベント情報をもとに、WordPressに記事を下書き投稿して！**
> 
> ・イベント名：春のモデルハウス見学会
> ・開催日時：4月XX日(土)〜XX日(日)
> ・場所：〇〇県〇〇市
> ・ターゲット：30代の子育て世帯
> ・アピールポイント：広い土間収納、家事動線、無垢材
> ・画像のヒント：ナチュラルテイストで明るいリビング

これらを送っていただければ、Antigravityが以下の処理を代行します。
1. ブログ用のHTML文章の生成
2. アイキャッチ画像の生成
3. WordPressへの下書き保存（ブラウザ操作 または API経由）

---

## 🛠️ トラブルシューティング
*   **AIエージェントの操作が途中で止まる場合:** セキュリティ画像認証（CAPTCHA）などが出た場合は手動で突破し、再度エージェントに続きを指示してください。画像のアップロードがうまくいかない場合（ドラッグ＆ドロップ不可など）は、生成された画像をダウンロードして手動でアップロードしてください。
*   **「エラー: .envファイルに...」と出る場合:** (API入力時) `.env` ファイルが設定されているか確認してください。
*   **「ステータスコード: 403/401」と出る場合:** (API入力時) セキュリティによるAPI制限や、アプリケーションパスワードの間違いが原因です。AIエージェントモード（方法1）をお試しください。
